Welcome to Memorizi! This guide will help you get started with using our platform, either by accessing the deployed site or setting up the project locally.

## Using the Deployed Site

You can start using Memorizi immediately by visiting our deployed site: [memorizii.vercel.app](https://memorizii.vercel.app/).
